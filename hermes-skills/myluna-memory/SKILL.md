# myLuna Memory

Use this skill when the user asks about past information that may already exist in myLuna memory.

Examples include:

- past meetings
- customer history
- product knowledge
- company decisions
- open house events
- previous discussions

## Search Command

To search memory, run:

```bash
myluna memory-search syspex "<query>"
